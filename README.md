# React Portfolio

A modern, responsive personal portfolio website built with React. Showcases projects, skills, and contact information.

![Portfolio Screenshot](./screenshot.png)

**🔗 Live Demo:** [View Live Demo](https://kevinwangombe.github.io/react-portfolio)

## ✨ Features

- Hero section with introduction
- Skills showcase
- Projects gallery
- About section
- Contact form
- Smooth scroll navigation
- Dark/Light mode (if implemented)
- Fully responsive

## 🛠️ Tech Stack

- React.js
- CSS3 / Styled Components
- JavaScript (ES6+)
- React Router (if used)

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/kevinwangombe/react-portfolio

# Install dependencies
npm install

# Start development server
npm start
```

## 📁 Project Structure

```
src/
├── components/    # Reusable UI components
├── pages/         # Page sections
├── styles/        # Global styles
└── App.jsx        # Main app component
```
